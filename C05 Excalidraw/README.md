# C05 Excalidraw

> **All Excalidraw drawings in the vault. Centralized.**

## What goes here

`.excalidraw.md` files — drawings made with the Excalidraw plugin.

## Plugin configuration

1. Community Plugin: **Excalidraw**
2. Settings → Excalidraw:
   - Excalidraw file folder: `C05 Excalidraw`
   - Default save location: `C05 Excalidraw`

That way new drawings land here automatically.

## Why centralize

- Links resolve by name, not path
- Easy to find all drawings
- Exporting to a new vault = 1 folder

## How to reference

From any note:

```markdown
![[ledger.excalidraw]]
```

(No path needed)

## Naming convention

- Thematic drawings: kebab-case — `database-schema-blindpay.excalidraw.md`
- Scratch/draft drawings: keep `Drawing YYYY-MM-DD HH.MM.SS.excalidraw.md` (plugin default)
