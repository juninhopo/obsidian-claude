# A00 Inbox

> **Entry point for EVERYTHING that arrives unclassified.**

## What goes here

- Loose ideas captured quickly
- Articles, links, screenshots, audio for triage
- Raw notes from mobile
- Anything you don't yet know where to place

## Golden rule

**Nothing stays in A00 for more than 7 days.** Once it lands here, it either moves to A01 (processing), is promoted directly to A02/B01/B03, or goes to B04/A05 (archive/backlog).

## Flow

```
input → A00 → (weekly triage) → A01/A02/B01/B03/A05/B04
```

## How to process

Invoke `/process-inbox` (skill at `C04 Claude Obsidian/02 Skills & Commands/process-inbox.md`). I read each item, propose a destination, and you approve.

## What does NOT go here

- Anything that already has a clear owner (daily notes go directly to A04, project items to B01, etc.)
- Automated downloads from integrations (Readwise goes to C02)

## Maintenance checklist

- [ ] Weekly triage (every Friday?)
- [ ] Nothing older than 7 days
- [ ] Each item has a clear destination after triage
