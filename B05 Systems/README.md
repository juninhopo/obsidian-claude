# B05 Systems

> **Your work systems. SOPs, workflows, recurring processes.**

## What goes here

- **Workflows** — how you do X (e.g., `code-review-process.md`)
- **SOPs** — standardized procedures (e.g., `deploy-checklist.md`)
- **Process templates** — (note templates go in C04/06)
- **Rituals** — personal/professional routines (weekly review, sprint planning)
- **Meta-rules** — how this vault operates (beyond CLAUDE.md)

## Difference from B03 Resources

- **B03** = knowledge (what you know)
- **B05** = operation (how you work)

E.g.:
- B03: `kubernetes-basics.md` (what it is)
- B05: `deploy-kubernetes-sop.md` (how I deploy)

## Difference from C04 Skills

- **B05 Systems** = systems YOU execute (personal/professional)
- **C04 Skills** = workflows the AI executes when you invoke them

## When to create

When you notice you **keep doing the same thing repeatedly**. A documented system reduces decision fatigue.

## Rule

**Minimalist.** Only document what you actually use. An SOP no one follows = junk.
