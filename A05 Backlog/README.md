# A05 Backlog

> **Parking lot. Undated ideas and tasks.**

## What goes here

- Project ideas you want to do "some day"
- Non-urgent tasks without a clear owner
- "If I ever have the time, I want to..."
- Pending items that do not fit any active project

## Difference from the others

- **A04 Daily Note** = task with a specific date
- **B01 Projects/_\<project\>.md** = task tied to a project
- **A05 Backlog** = NO date, NO project yet

## Suggested format

Single files or a single `backlog.md` divided into sections:

```markdown
## Project ideas
- [ ] Build a bot that...

## To study one day
- [ ] Rust
- [ ] Distributed systems design

## Shopping list / experiments
- [ ] Try Zed editor
```

## Rule

**Quarterly review.** An item sitting for 1 year in the backlog = decide:
- promote to active project (B01)
- archive (B04) with honest "I will not do this"

Without this review, the backlog becomes a graveyard.

## Checklist

- [ ] Quarterly review
- [ ] Old items: promote or archive
