# A04 Daily Notes

> **Your daily journal. Time-based capture.**

## What goes here

- 1 file per day — `YYYY-MM-DD.md`
- Day log, meetings, decisions, quick ideas
- Links to projects/anchors relevant on that day

## Obsidian configuration

Enable the core plugin **Daily Notes**:
1. Settings → Core Plugins → Daily Notes (ON)
2. Settings → Daily Notes:
   - Date format: `YYYY-MM-DD`
   - New file location: `A04 Daily Notes`
   - Template file location: `C04 Claude Obsidian/06 Templates & References/template-daily-note`

## Structure (template)

Use `C04 Claude Obsidian/06 Templates & References/template-daily-note.md`

## Bronze rule

A daily note **is not permanent**. It is a draft. Ideas that matter migrate to A01/A02/B03 at the end of the week.

## How to create

- Option 1: `Cmd+P` → "Daily notes: Open today's daily note"
- Option 2: Skill `/create-daily`

## Archival

Daily notes older than 6 months may go to `B04 Archives/daily-notes-archive/` (manual). But it is fine to keep them here — they are small.
