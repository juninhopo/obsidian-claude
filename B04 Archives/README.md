# B04 Archives

> **Dead archive. What is not alive but we do not delete.**

## What goes here

- Old projects that closed (move from B01)
- B03 notes that became false/obsolete
- Daily notes older than 6 months (optional)
- Legacy backups from old tools (Notion exports, etc.)
- **credentials-private/** — sensitive data, isolated

## Rule

**Nothing is deleted in the vault.** If something stops being useful, it comes here.

## When to move here

- B01 project closed (no return) → move to B04/old-projects/
- B03 note became false/obsolete → move to B04/obsolete-resources/
- Daily notes older than 6 months → move to B04/daily-notes-archive/
- Backups of old tools → dedicated folder

## Difference from "delete"

Archiving preserves history. Deleting erases knowledge. The AI has access to the archive to search historical context when necessary — "2 years ago you decided X because of Y".

## Suggested structure

```
B04 Archives/
├── old-projects/             (closed B01 projects)
├── obsolete-resources/       (B03 notes that became false/outdated)
├── daily-notes-archive/      (daily notes older than 6 months, optional)
├── credentials-private/      (sensitive data — isolated, NEVER in public notes)
└── <your-categories>/
```

*Don't actually create empty folders — they're conceptual buckets you fill as needed.*

## ⚠️ credentials-private

Credentials stay isolated here. **NEVER reference them in public notes.** If they need to be used, the AI can read and use them without exposing content in output.
