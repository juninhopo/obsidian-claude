# A01 Processing

> **Intermediate work zone. Ideas being refined between capture and publication.**

## What goes here

- Notes that came from A00 but are not yet mature enough for A02 (Anchor Topic) or B03 (Resource)
- Drafts of authored notes under construction
- Fragments being connected before becoming something permanent
- Zettelkasten in fluid state

## Difference from A00

- **A00** = "I don't know what this is yet, I haven't triaged it"
- **A01** = "I know what it is, I'm working on it"

## Flow

```
A00 → A01 (refinement) → A02 Anchor Topics OR B03 Resources
```

## Rule

A note in A01 has:
- A clear title
- A defined central idea
- No links to the rest of the vault yet (if it had them, it would already be in A02)

## When to promote

- Became a clear thesis → **A02 Anchor Topics** (as MOC)
- Became a consolidated reference/study → **B03 Resources**
- Didn't move forward → **A05 Backlog** (parking lot)

## Checklist

- [ ] Biweekly review of items in A01
- [ ] Promote or archive — do not let them stagnate
